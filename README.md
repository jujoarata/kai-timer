# 界タビ20s 予約開始日計算機

星野リゾート「界タビ20s」の予約開始日を計算するシンプルなツールです。

## 🎯 概要

界タビ20sは、チェックイン日の**44日前の午前0:00（深夜24時）**から予約が開始されます。このツールでは、希望のチェックイン日を入力するだけで、予約開始日を自動計算します。

## 🚀 使い方

1. [ツールを開く](https://jujoarata.github.io/kai-timer/)
2. チェックイン希望日を選択
3. 予約開始日が表示されます

## ⚠️ 注意

- 予約開始は**前日の深夜24時（＝当日の0:00）**です
- 人気の日程は争奪戦になるため、前日の夜から待機することをおすすめします

## 📱 対応環境

- スマートフォン・PC両対応
- モダンブラウザ推奨（Chrome、Safari、Firefox、Edge）
